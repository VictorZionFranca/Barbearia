Funcionamento do Agendamento

Para criar um agendamento, clique em um horário desejado no calendário. Preencha os campos necessários no formulário.

Após preencher as informações, clique no botão Salvar para confirmar o agendamento.

Clique uma vez na tarefa para marcar como concluído.

Clique duas vezes na tarefa para excluir o agendamento.

Após marcar um agendamento como concluído, um clique único exibirá a opção de excluir o agendamento.
